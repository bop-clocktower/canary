how it works

Natural language
→ production tests.

Describe what to test. Canary analyzes your codebase, infers fixtures, and generates framework-appropriate tests — with the Clocktower voice system keeping output consistent.

analyze generate review migrate
terminal 
# generate tests for a component
$ canary generate "test the login flow"

# canary analyzes, then generates:
◆  canary__analyze_file    login.ts
◆  canary__generate_test   playwright

# migrate a whole suite
$ canary migrate --from jest --to vitest

✓  0 failures
