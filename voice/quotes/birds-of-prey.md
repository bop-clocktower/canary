## Contribution process

- **One row per PR.** The PR description must include the citation (issue +
  year + page) and ideally a panel screenshot or scan reference.
- PRs adding rows **without** a primary-source citation are rejected.
- **LLM-generated quote suggestions are rejected on sight**, regardless of how
  canon-shaped they read.
- Re-verification is welcomed: if a previously-accepted row is later found to be
  misattributed, it is **struck through** (not silently removed) so the change
  stays auditable.

## Trustworthy sourcing paths

In rough order of reliability — starting points, not pre-approved sources:

1. **Wikiquote** — citation-discipline-focused; surviving entries are
   issue-attributed. Verify against primary anyway.
2. **DC Database wiki** — typically issue-and-page-attributed; verify against
   primary.
3. **Primary issues** via DC Universe Infinite / Hoopla / owned trades — the
   gold standard.
4. **Birds of Prey Omnibus / Compendium** trade collections.
